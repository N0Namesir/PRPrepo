## 📋 Descripción

Repositorio colaborativo diseñado para practicar y dominar **Git** y **GitHub** en un entorno real de trabajo en equipo.

## 🎯 Objetivo Principal

Demostrar una colaboración efectiva entre dos desarrolladores trabajando en dispositivos diferentes, implementando un flujo de trabajo profesional con ramas y pull requests.

## 📊 Indicadores de Logro

- ✅ Modificar archivos en ramas personales
- ✅ Subir cambios a repositorio remoto
- ✅ Crear pull requests hacia rama `develop`
- ✅ Revisar y aprobar cambios colaborativamente
- ✅ Analizar resultados y documentar aprendizajes

## 🚀 Flujo de Trabajo

1. Crear rama personal desde `develop`
2. Realizar cambios y commits descriptivos
3. Pushear cambios a la rama remota
4. Abrir pull request para revisión
5. Fusionar cambios aprobados
6. Actualizar rama local

## 📚 Tecnologías

- Git
- GitHub
- Markdown
